## 🔧 推送渠道管理

### 查看推送状态
```bash
# 查看定时任务执行历史
openclaw cron runs <任务ID>

# 查看推送状态
# 状态包括: pending, delivered, failed, unknown
```

### 修改推送配置
如果需要修改推送配置，可以更新定时任务的delivery参数:

```bash
# 修改推送模式
openclaw cron update <任务ID> --delivery.mode announce

# 修改推送渠道
openclaw cron update <任务ID> --delivery.channel wecom

# 修改推送目标
openclaw cron update <任务ID> --delivery.to <用户ID>
```

### 推送模式选项:
1. **announce** - 系统事件推送 (当前使用)
2. **none** - 不推送，仅保存结果
3. **webhook** - Webhook推送 (需要配置URL)

### 推送渠道选项:
1. **wecom** - 企业微信 (当前使用)
2. **telegram** - Telegram
3. **discord** - Discord
4. **webchat** - Web聊天界面
